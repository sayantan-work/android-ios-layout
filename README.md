# EasyDo 365 — iOS Skeleton

Glassmorphic skeleton for the iOS port of EasyDo 365.

Open index.html in any browser. Pick a persona, click tabs in the bottom bar to switch.

## Tab Bar (3 tabs, same for everyone)
- Home
- Workspace (Tasks + Meets + Notes + Reminders)
- Chat

## Nav Bar
- Company switcher + Branch chip (HRMS users)
- Notifications bell
- Avatar (tap = open Me)

## Personas
- A. App User (productivity-only, no company)
- B/C. HRMS Regular Employee / L3
- D. HRMS L2 (Branch Manager)
- E. HRMS L1 (CEO)
