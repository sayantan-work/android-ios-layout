EasyDo 365 — v18 (Stacks)

The first iteration that's only the punch page, fully fledged across 4 states. v1–v17 had one giant circle with the time in it; v18 ships a stack of modular squircle tiles — hero · location · hours ring · timeline · team · up-next — where the hero tile morphs by state (pre-punch CTA → live timer → day summary). Lineage: iOS widget gallery × Apple Sports × Apple Watch Modular face × Arc Search cards.

Single accent is warm coral #FF6B35 — untouched in v1–v17 (every prior version used violet, cobalt, red, amber, or emerald). Tiles are 28px radius (vs v17's 14px), flat surfaces with 1px hairline + inner top highlight. Hero tile gets a soft coral glow when state-active. Cycle states with the STATE chip in the prototype.

v18 hero idea

  • One screen, four states. Hero morphs: CTA → live timer → day summary.
  • Stack of tiles, not one big circle. Widget-gallery vocabulary.
  • Coral primary — first v1–v18 version to use coral as brand accent.

The 6 tiles

  • Hero — primary state, big number, primary action.
  • Location — mini-map + geofence status.
  • Hours ring — progress vs target, break segments.
  • Today's timeline — horizontal scrubber 09:00 → 18:00.
  • Team — who's in, who's on leave.
  • Up next — meeting, lunch, branch close.

State recipe

  • PRE · coral "Punch In" CTA, empty ring, planned shift highlight.
  • MOMENT · sheet with face capture + geofence confirm.
  • MID · live HH:MM:SS timer (counts up every second), break/punch-out actions.
  • POST · emerald check, total + overtime breakdown, regularize action.

Tile shape

  • 28px radius (vs v17's 14px) — squircle vocabulary.
  • 1px hairline border rgba(255,255,255,0.06).
  • Inner top highlight 1px rgba(255,255,255,0.04) — depth without shadow.
  • Hero only: coral outer glow when state-active.

Carries from v17

  • 5-tab IA, single identity, permission model.
  • Light/dark toggle (key: v18-mode).
  • Status bar + org chip + bell header.
