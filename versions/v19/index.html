EasyDo 365 — v19 (Quiet)

The minimalist counter-iteration to v18. Where v18 stacked six squircle tiles, glowing rings, coral CTAs, and live chips — v19 strips all of that down to the minimum that still answers "what's my punch state?". No tiles, no cards, no borders. Hairline rules and type carry every bit of hierarchy.

The only colour in the entire app is a single 6×6px coral dot that appears next to "Live" when punched in — everything else is monochrome. Display number is 88px Inter Light, big numbers are JetBrains Mono. Sections are separated by 48px of whitespace, not boxes. Lineage: Apple Health single-stat · iA Writer · Patagonia · Stripe Press.

v19 hero idea

  • No tiles, no cards. Hairlines + type only.
  • No colour except one coral dot when live.
  • One primary action per state, rendered as underlined text — never a filled button.

What was removed from v18

  • 6 squircle tiles → flat sections.
  • Coral filled CTAs → underlined text links.
  • Hours ring SVG → 1 number.
  • Geofence mini-map → 1 line.
  • Team avatars · Up-next list · all badges and chips.

Type recipe

  • Display: 88px Inter Light, letter-spacing -0.04em.
  • Big numbers: 64px JetBrains Mono.
  • Actions: 17px sans, 1px underline.
  • Labels: 10px mono uppercase, 0.18em tracking.

The four states

  • pre · clock + Punch in link + branch line.
  • moment · sheet with thin scan square + 3 hairline rows.
  • mid · coral live dot + 64px elapsed timer + Break/Punch out links.
  • post · check glyph + 8h 24m total + Regularize link.
